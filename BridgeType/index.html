What the app does

A translator built for real chat workflows.

Compose mode

Type in the keyboard draft, translate, preview the result, and insert it into the focused messenger text field.

Receive mode

Paste or dictate incoming text in the keyboard, then translate it without changing the outgoing message box.

Live mode

Translate while typing and insert translated text through the normal Android input connection.

Selected text

Translate text explicitly selected or shared by the user through Android ACTION_PROCESS_TEXT or ACTION_SEND.

Voice translation

Start speech recognition from the mic button, control voice timing settings, and translate recognized text.

Themes and languages

Use language-specific layouts, saved language pairs, keyboard themes, color options, emoji, and ready phrases.

Privacy-first design

BridgeType avoids unsafe messenger access patterns.

No chat screen inspection

The app does not read full chat screens, scrape chat bubbles, use overlays, or modify other messenger apps.

No clipboard monitoring

Clipboard access is only tied to explicit user actions such as pressing paste or copy. The app does not listen for clipboard changes.

Keyboard text only

The keyboard processes the text the user types, dictates, pastes, or explicitly sends to BridgeType for translation.

On-device translation

After language models are downloaded, ML Kit translation runs on the device. Messages are not stored by BridgeType.
