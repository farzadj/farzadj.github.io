Common support topics

  • App install or update problems from Google Play or local builds
  • Offline deck import and milestone progress behavior
  • Text-to-speech playback or device voice availability
  • Pro subscription access and purchase restore behavior
  • Flashcard corrections, missing translations, or formatting issues

Support information to include

When reporting a bug, include the app version, Android version, device model, and the exact screen or action that caused the issue.

If the issue relates to content, include the level, content type, deck or milestone, and the card text if possible.

Contact channel

FlashDeutsch currently uses GitHub as the primary public support channel.

  • Repository: github.com/farzadj/FlashDeutsch
  • Issues: github.com/farzadj/FlashDeutsch/issues
