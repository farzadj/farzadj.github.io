📱 Forex Lab Android App

Native Android app with separate pages for forex trading analysis and strategy management.

📊 Native Pages

  • Chart page: native candlestick chart + strategy zones + signal markers
  • Data page: backend live feed settings
  • Settings page: strategy and detector parameters
  • Signals page: native RecyclerView signal list
  • Help page: in-app strategy and risk guide

🌍 Live Data Feed

Backend Integration: Live forex data is fetched by the backend from FCS API

Architecture: Android talks only to the Forex Lab backend

Data Page Controls:

  • pair - Select forex pair
  • interval - Candlestick interval
  • refresh seconds - Live polling frequency
  • bars fetched - Number of historical bars
  • Update live polling against the backend

⚙️ Technology Stack

Language & Framework: Kotlin + Fragments + BottomNavigation

State Management: Shared TradingViewModel for app-wide state

Graphics: Native custom chart rendering

UI Components: RecyclerView for signals, native Android components
